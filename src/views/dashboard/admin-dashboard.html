Bienvenido al Panel de Administrador

En este espacio puedes gestionar usuarios, ver reportes y configurar el sistema.

Funciones del Administrador

  • Gestionar información de usuarios y permisos.
  • Visualizar y generar reportes.
  • Configurar opciones del sistema y ajustes generales.

Acciones Disponibles

Puedes comenzar a gestionar el sistema desde el menú desplegable que aparece en la parte superior.
